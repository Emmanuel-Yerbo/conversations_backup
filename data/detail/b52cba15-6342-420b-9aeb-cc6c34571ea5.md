### 🤖 Gemini/Antigravity — *2026-05-20 12:16:40*

To toggle between the **Editor** and the **Agent Manager (Full Agent Workspace)** in Google Antigravity, you can use any of the following methods:

### 1. Interactive UI Buttons (Alternative to the Shortcut)
* **To open the Agent Manager:** Click the **Open Agent Manager** button in the top-right corner of the IDE's top menu/status bar.
* **To return to the Editor:** Click the **Open Editor** button, also located in the top-right corner.

---

### 2. Why `Ctrl + E` might not be working:
* **Focus Conflict:** If your cursor is currently active in a terminal pane, `Ctrl + E` might be intercepted by shell emulators (which commonly map `Ctrl + E` to *"move cursor to end of line"*). Click inside a file or empty space in the editor and try again.
* **Keyboard Shortcut Collisions:** Another extension or user setting may have overridden the shortcut.

---

### 3. Check/Fix via the Command Palette
1. Open the Command Palette using **`Ctrl + Shift + P`** (or `Cmd + Shift + P` on macOS).
2. Type **`Toggle Agent Manager`** or **`Antigravity: Open Agent Manager`** and press **Enter** to run it.
3. To rebind it to a working shortcut:
   * Open the keyboard shortcuts settings (search for `Preferences: Open Keyboard Shortcuts` in the Command Palette).
   * Search for `Toggle Agent Manager` or `Antigravity`.
   * Double-click the command to bind it to `Ctrl + E` (or any other combination of your choice).
